# 💰 个人记账系统

一个基于 C++ 的命令行个人记账工具。

## ✨ 功能特性

- 📝 添加/删除收入支出记录
- 📊 月度收支统计报表
- 📈 消费分类统计（餐饮、购物、交通等）
- 💰 月度预算设置与提醒
- 🔍 备注关键词搜索
- 💾 数据自动保存到本地文件

## 🛠 技术栈

- C++11
- STL（vector, algorithm, fstream）
- 面向对象设计（封装、继承、多态）
- 文件持久化存储

## 📁 项目结构
PersonalFinance/
├── Bill.h/cpp # 账单类
├── Account.h/cpp # 账本管理类
├── main.cpp # 主程序入口
└── data/ # 数据存储目录
